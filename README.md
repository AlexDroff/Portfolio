# Portfolio graficzne i DTP - Oleksandr Aleksandrov

## Opis
Polskojezyczna strona portfolio dla projektanta graficznego, specjalisty skladu DTP i przygotowania do druku.

## Pozycjonowanie
Portfolio prezentuje prace z obszaru identyfikacji wizualnej, materialow drukowanych, katalogow, plakatow, wizytowek, okladek ksiazek, opakowan oraz prepress.

## Glowne sekcje
- Profil i dane kontaktowe
- Kompetencje techniczne zwiazane z grafika, skladem i DTP
- Kategorie projektow
- Galerie prac

## Technologie wdrozeniowe
Projekt jest zbudowany jako statycznie generowana aplikacja Next.js.

## Zasoby
Materialy graficzne uzywane przez aktualna strone znajduja sie w katalogu `public/work/`, a zyciorys w pliku `public/pdf/CV_Oleksandr_Aleksandrov.pdf`.
