# Portfolio graficzne i DTP - Oleksandr Aleksandrov

## Opis
Statyczna, polskojęzyczna strona portfolio dla projektanta graficznego, specjalisty składu DTP i przygotowania do druku.

## Pozycjonowanie
Portfolio prezentuje prace z obszaru identyfikacji wizualnej, materiałów drukowanych, katalogów, plakatów, wizytówek, okładek książek, opakowań oraz prepress.

## Główne sekcje
- Profil i dane kontaktowe
- Kompetencje techniczne związane z grafiką, składem i DTP
- Zakres projektów
- Kolekcja prac w sliderach

## Technologie wdrożeniowe
Projekt jest zbudowany jako statyczna strona z użyciem HTML, CSS i vanilla JavaScript. Technologie te opisują wyłącznie sposób wykonania strony.

## Zasoby
Materiały graficzne znajdują się w katalogu `images/`, a życiorys w pliku `pdf/CV_Oleksandr_Aleksandrov.pdf`.
